# Projetos de Introducao a Ciencia de Dados

Este repositorio reune atividades praticas desenvolvidas na disciplina de Introducao a Ciencia de Dados, com foco em Python, `pandas`, limpeza de dados, analise exploratoria e visualizacao.

Cada pasta representa um projeto ou atividade da disciplina.

## Estrutura do repositorio

- `analise-vgsales/`: analise exploratoria da base Video Game Sales, com 7 perguntas respondidas em notebook. O dataset e baixado via `kagglehub`.
- `pandas_intro/`: atividade introdutoria de exploracao de dados com comandos basicos do `pandas`.
- `aula_pandas2/`: atividade de limpeza, transformacao, padronizacao e agrupamento de dados.
- `medidas_visualizacoes/`: exercicios com metricas e visualizacoes a partir de dados do Bolsa Familia.
